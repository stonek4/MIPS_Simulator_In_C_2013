The purpose of this program is to simulate the following MIPS commands in C:

LUI
OR
ORI

As well as:

Multiplication and division using bit-wise shift.
Converting numbers to 16-bit binary patterns
Manually managing memory

To run, simply compile the main.c file and run using two arguments, an input and output file (output file can be anything)

./a.out input.txt output.txt

You can use numbers.txt as an example input file.
